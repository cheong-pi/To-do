# 자동 테스트 체크리스트

```powershell
npm test
npm run build
```

- 저장된 유효한 알림 시간이 앱 재실행 후 유지되는지 검사한다.
- 손상된 항목이 앱 전체를 멈추지 않고 제외되거나 기본값으로 복구되는지 검사한다.
- 불가능한 날짜와 종료일이 시작일보다 빠른 일정이 로드되지 않는지 검사한다.
- 백업 복원 및 삭제 전 자동 복구본이 전체 저장 키를 되돌리는지 검사한다.
- 오늘 할일 정렬 순서가 제품 규칙과 일치하는지 검사한다.

현재 기준은 테스트 파일 6개, 시나리오 18개다. 새 저장 필드나 날짜 규칙을 추가하면 대응하는 회귀 테스트를 함께 추가한다.
